## 删除文件或文件夹

* 路径: ```/v2/user/delete```
* 方法: ```POST```
* 登录: 需要

#### 需要提交的字段


文件路径的的数组

#### 注意事项

* 只接受文件路径，如果要使用文件ID删除，请使用下一个接口

#### 示范

* 发送

```POST``` ```/v2/user/delete```

```json
[
  "/疯狂的石头",
  "/CentOS7",
]
```

* 接收：

```json
{
    "status": 0,
    "result": {
        "data": 3
    },
    "code": "OK",
    "success": true
}
```

* data 字段用以指示成功删除的文件数量。
